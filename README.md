# 류성열 | Backend Developer 

안녕하세요.   
핀테크 스타트업에서 B2C 서비스를 개발하고 있는 Kotlin, Spring Boot 기반의 백엔드 개발자 류성열입니다.

### 🚀 AI-Driven Development
AI를 단순한 코드 생성 도구가 아니라, 반복 업무를 줄이고 문제 해결 속도를 높이는 개발 파트너로 활용합니다.       
스타트업에서 빠른 출시와 잦은 변화에 대응하며, AI 주도 개발로 3개월 만에 선불 카드 시스템 초기 모델을 출시했습니다.    

### 🧭 근거 기반으로 판단하는 개발자
AI를 활용한 개발에서 중요한 것은 더 많은 코드를 생성하는 것이 아니라, 생성된 결과가 왜 타당한지 판단하는 능력이라고 생각합니다.   
Plan First Workflow와 Human Approval Gate를 적용해 AI의 제안을 사전에 검토하고, 사람이 최종 판단하는 개발 흐름을 설계했습니다.

구현은 자동화될 수 있어도 판단은 개발자의 책임입니다.  
근거 있는 판단으로 빠른 개발과 안정적인 제품 출시를 함께 만드는 엔지니어가 되겠습니다.
